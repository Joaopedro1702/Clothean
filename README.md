# 🧺 Clothean — Lavanderia Online

Site institucional e de reservas da **Clothean**, uma lavanderia online com entrega em até 24 horas.

---

## 📋 Sobre o Projeto

A Clothean é uma lavanderia online que oferece serviços de lavagem, secagem e passamento de roupas com coleta e entrega na porta do cliente. Este projeto é o site oficial da empresa, desenvolvido com HTML, CSS e JavaScript puro.

---

## 🗂️ Estrutura de Páginas

| Página | Descrição |
|---|---|
| `index.html` | Página principal com apresentação, serviços, formatos e localização |
| `login.html` | Página de login e cadastro do usuário |
| `reserva.html` | Formulário de reserva de serviço com resumo do pedido |

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização e responsividade
- **Bootstrap 5.3.8** — componentes (carousel, offcanvas, formulários)
- **JavaScript** — interações do menu e do carousel

## 📱 Responsividade

O site é responsivo e se adapta a três tamanhos de tela:

- **Desktop** — layout completo com duas colunas
- **Tablet** — até 900px, elementos reorganizados em coluna
- **Mobile** — até 600px, menu hamburguer, fontes reduzidas

## 🚀 Como Rodar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/Clothean1.5.git
```

2. Abra a pasta do projeto:
```bash
cd Clothean1.5
```

3. Abra o `index.html` no navegador ou use um servidor local como o **Live Server** do VS Code.
